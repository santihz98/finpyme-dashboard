# FinPyme Dashboard
> Dashboard financiero con IA para pymes colombianas

## Stack
Next.js 14 · TypeScript · Tailwind CSS · Recharts · FastAPI

## Repositorios relacionados
- finpyme-backend (FastAPI + PostgreSQL)
- finpyme-mock-generator (Python data generator)

## Setup local
1. cp .env.example .env.local
2. Agrega NEXT_PUBLIC_API_URL=http://localhost:8000
3. npm install
4. npm run dev
